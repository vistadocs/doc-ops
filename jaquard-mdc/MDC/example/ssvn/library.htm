Structured System variables -- ^$LIBRARY

^$LI[BRARY]

Approved for inclusion in a future ANSI M[UMPS] language standard.

This structured system variable provides information about the library functions that are currently available to the M[UMPS] system.

WRITE !,"The following libraries are available:"
SET E="ELEMENT"
SET L="" FOR  SET L=$ORDER(^$LIBRARY(L)) QUIT:L=""  DO
. WRITE !,L . SET F="" FOR  SET L=$ORDER(^$LIBRARY(L,E,F)) QUIT:F=""  DO
. . WRITE !," - Function ",F
. . QUIT
. QUIT
