Structured System variables -- ^$LOCK

^$L[OCK]

Introduced in the 1995 ANSI M[UMPS] language standard.

This structured system variable provides information about the LOCKs that are currently known to the M[UMPS] system.

WRITE !,"Contents of the LOCK table:"
SET L="" FOR  SET L=$ORDER(^$LOCK(L)) QUIT:L=""  DO
. WRITE !,L
. QUIT

Approved for addition in a future ANSI M[UMPS] language standard.

If your implementation has granted you the privilege to do so, you may be able to remove an entry from the LOCK table by executing the command
KILL ^$LOCK(name)
