Structured System variables -- ^$ROUTINE

^$R[OUTINE]

Introduced in the 1995 ANSI M[UMPS] language standard.

This structured system variable provides information about the routines that are currently known to the M[UMPS] system.

WRITE !,"Routine directory:"
SET R="" FOR  SET R=$O(^$ROUTINE(R)) QUIT:R=""  DO
. WRITE !,R
. SET C=$GET(^$ROUTINE(J,"CHARACTER"))
. IF C'="" WRITE ?30,"character set is """,C,"""."
. QUIT

Approved for addition in a future ANSI M[UMPS] language standard.

If your implementation has granted you the privilege to do so, you may be able to remove a routine from the system by executing the command
KILL ^$ROUTINE(name)
