M[UMPS] Commands

H[ALT]

Introduced in the 1977 ANSI M[UMPS] language standard.

READ "Password: ",X
IF X'=^SECRET(+$HOROLOG) HALT
Don't allow an intruder any access to the system.

IF ^ERRCODES(ERROR)["DISK HARDWARE FAILURE" HALT
Get out before any physical harm is caused to the database.
