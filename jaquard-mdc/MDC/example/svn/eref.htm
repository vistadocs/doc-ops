Special variables -- $EREF

$ER[EF]

Introduced in the 1995 ANSI M[UMPS] Windowing Application Programmer's Interface standard.

The value of this special variable is the name of the node in ^$WINDOW that was involved in the most recent error that occurred.

Assuming that there is no button defined with the name "something wrong",
SET ^$WINDOW("DEMO","DEFBUTTON")="something wrong"
will cause an error with $ECODE containing ",M52," and $EREF equal to the canonic form of the name of the above reference to ^$WINDOW.
