Special variables -- $TLEVEL

$TL[EVEL]

Introduced in the 1995 ANSI M[UMPS] language standard.

The value of this variable identifies the current depth of nesting of TSTART commands (but... keep in mind, there is only at most one transaction active at any given time).

IF $TLEVEL WRITE !,"Don't nest these activities" QUIT
TSTART *:TRASACTIONID="MONOPOLY"
Every TSTART command increments the value of special variable $TLEVEL (initial value is 0).
