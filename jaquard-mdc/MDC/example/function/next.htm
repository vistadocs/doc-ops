M[UMPS] Functions - $N[EXT]

Introduced in the 1977 ANSI M[UMPS] language standard; removed in the 1995 ANSI M[UMPS] language standard.
This function is obsolete. Use the function $ORDER instead.

Be careful when converting code from using the function $NEXT to using $ORDER: code fragments referring to loopcounter<0 and loopcounter=-1 may occur several lines past the 'end' of the loop.
