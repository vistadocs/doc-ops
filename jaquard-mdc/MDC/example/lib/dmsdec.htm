Library Functions -- $%DMSDEC^MATH

Approved for inclusion in a future ANSI M[UMPS] language standard as part of the library for General Mathematics.

This function returns the value in (decimal) degrees of the value of its parameter (in degrees, minutes and seconds).

Reference   Value
$%DMSDEC^MATH("90:00:00")   90
$%DMSDEC^MATH("90:30:00")   90.5
$%DMSDEC^MATH("90:30:10")   90.5027777777777...

The MDC has approved code to approximate the return value of this function. Implementors are encouraged to provide more accurate and efficient code). Click here to view or download the code for this function. Click here to view or download the code for all M[UMPS] Library Functions.
