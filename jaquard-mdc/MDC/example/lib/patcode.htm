Library Functions -- $%PATCODE^STRING

Approved for inclusion in a future ANSI M[UMPS] language standard as part of the library for miscellaneous character functions.

This function returns a boolean value that indicates whether or not a string matches a pattern code in a certain character set.

The actual entry point that is called for this function can be obtained through ^$CHARACTER(charset,"PATCODE",patcode)=entrypointname.

The MDC has approved code to approximate the return value of this function. Implementors are encouraged to provide more accurate and efficient code). Click here to view or download the code for this function. Click here to view or download the code for all M[UMPS] Library Functions.
