Library Functions -- $%COMPARE^CHARACTER

Approved for inclusion in a future ANSI M[UMPS] language standard as an override mechanism for collating strings.

This function returns a value that indicates the order in which two strings collate, when compared in the specified environment. The environment specified may be:

  • absent, in which case the collating scheme for the current process prevails
  • the name of a global variable, in which case the collating scheme for that global variable will be used
  • the name of a character set profile, in which case the collating scheme for that character set profile will be used

Reference   Value
SET A="abc",B="ABC"    
$%COMPARE^CHARACTER(A,B)   1
$%COMPARE^CHARACTER(A,B,"^P")   0
